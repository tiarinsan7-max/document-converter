# React Web UI

Modern web interface for Universal Document Converter.

## Setup

```bash
cd web
npm install
```

## Development

```bash
npm start
```

Runs on http://localhost:3000

## Build

```bash
npm run build
```

Creates optimized production build in `build/` directory.

## Features

- Drag & drop file upload
- Single and batch conversion
- Real-time progress tracking
- Download converted files
- Responsive design
- Modern UI with Chakra UI

## Technology

- React 18
- Chakra UI
- Axios
- React Dropzone
- React Icons
